# 🚀 IoT Security Scanner (SOC Dashboard)

A full-stack cybersecurity platform that detects exposed internet-connected devices using Shodan API and analyzes risks.

## 🔐 Features
- JWT Authentication (Role-Based)
- Real-time IoT Device Scanning
- Risk Scoring Engine (0–100)
- SOC-style Dashboard with Logs
- Filters, Pagination & Search
- Analytics with Charts

## 🛠 Tech Stack
- Backend: Flask, SQLAlchemy
- Frontend: React, Tailwind CSS
- API: Shodan
- Auth: JWT
